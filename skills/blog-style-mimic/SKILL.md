# Blog Style Mimic

사용자의 글쓰기 스타일을 분석하고, 새로운 글을 같은 말투로 작성한다.

## When to Use

- 사용자가 자신의 말투로 글을 써달라고 요청할 때
- 블로그 글, SNS 포스트 등에서 일관된 어투가 필요할 때
- 기존 글의 톤을 유지하면서 새로운 내용을 작성할 때

## How It Works

### 1단계: 샘플 수집

`contexts/my-blog-style.md`에서 사용자가 등록한 말투 샘플을 읽는다.

최소 요구사항:
- 샘플 텍스트 300자 이상
- 가능하면 2개 이상의 서로 다른 글

### 2단계: 말투 프로필 생성

다음 요소를 분석하여 말투 프로필을 만든다:

#### 어투 (Tone)
- **존댓말 유형**: 해요체 / 합니다체 / 혼합
- **반말 유형**: ~다 체 / ~임 체 / 구어체
- **격식 수준**: 격식(formal) / 반격식(semi-formal) / 캐주얼(casual)

#### 문장 구조 (Sentence Structure)
- **평균 문장 길이**: 짧음(~15자) / 보통(15~30자) / 김(30자+)
- **문장 연결 방식**: 짧게 끊기 / 접속사 연결 / 혼합
- **단락 길이**: 1~2문장 / 3~5문장 / 긴 단락

#### 어휘 특성 (Vocabulary)
- **줄임말 사용**: 높음(ㅋㅋ, ㅎㅎ, ㅠㅠ, ㄹㅇ) / 보통 / 없음
- **외래어/영어 혼용**: 빈번 / 가끔 / 거의 없음
- **전문용어 수준**: 전문적 / 일상적 / 쉬운 설명 위주

#### 감정 표현 (Emotional Expression)
- **이모티콘**: 사용함(😊🥰) / 이모티콘 대신 텍스트(ㅋㅋ) / 사용 안 함
- **감탄사**: 많음(와, 대박, 진짜) / 보통 / 거의 없음
- **강조 방식**: 느낌표!!! / 물결표~~~ / 대문자 / 볼드 / 없음

#### 문장 끝 패턴 (Sentence Endings)
- 주로 사용하는 어미 패턴을 3~5개 추출
- 예: "~했어요", "~거든요", "~인 것 같아요", "~더라고요"

### 3단계: 말투 프로필 출력 형식

```
## 말투 프로필

- 어투: 해요체, 캐주얼
- 문장 길이: 짧은 편 (평균 15~20자)
- 줄임말: ㅋㅋ, ㅎㅎ 가끔 사용
- 이모티콘: 거의 안 씀
- 문장 끝: ~했어요, ~거든요, ~더라고요, ~인듯!
- 단락: 2~3문장씩 짧게 끊음
- 특징: 구어체 느낌, 중간중간 독자에게 말 걸듯이 씀
```

### 4단계: 적용

새 글을 작성할 때 프로필의 모든 요소를 일관되게 적용한다.

## 일관성 검증 체크리스트

글 작성 후 다음을 확인한다:

- [ ] 어투가 처음부터 끝까지 동일한가 (해요체로 시작했으면 끝까지 해요체)
- [ ] 문장 길이가 샘플과 비슷한 범위인가
- [ ] 줄임말/이모티콘 사용 빈도가 샘플과 유사한가
- [ ] 문장 끝 패턴이 샘플에서 추출한 것과 일치하는가
- [ ] 단락 구분 방식이 샘플과 비슷한가
- [ ] 샘플에 없는 새로운 말투 습관이 섞이지 않았는가

## Examples

### Input (샘플)
```
오늘 카페 투어 다녀왔는데요 ㅎㅎ
성수동에 새로 생긴 카페인데 분위기가 진짜 좋더라고요~
라떼 시켰는데 거품이 엄청 부드러워서 깜짝 놀랐어요!
가격은 좀 있는 편이긴 한데... 분위기값 한다고 생각하면 괜찮은 것 같아요
다음에 또 가볼 예정입니다 ㅎㅎ
```

### Output (프로필)
```
- 어투: 해요체, 캐주얼
- 문장 길이: 보통 (20~30자)
- 줄임말: ㅎㅎ 사용
- 이모티콘: 미사용
- 문장 끝: ~는데요, ~더라고요, ~었어요, ~것 같아요, ~입니다
- 단락: 문장마다 줄바꿈
- 특징: 물결표(~) 가끔 사용, 말줄임표(...) 사용, 독백 느낌
```

### Output (같은 말투로 쓴 새 글)
```
오늘 홍대 쪽에 새로 오픈한 브런치 카페 다녀왔는데요 ㅎㅎ
인스타에서 보고 궁금했던 곳이었거든요~
에그 베네딕트 시켰는데 소스가 진짜 맛있더라고요!
양도 꽤 많아서 배부르게 먹었어요
가격이 18000원이라 좀 비싼 편이긴 한데... 맛 생각하면 나쁘지 않은 것 같아요
주말에 가면 웨이팅 있을 수 있으니까 평일에 가보시는 걸 추천합니다 ㅎㅎ
```
