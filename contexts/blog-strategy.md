# 블로그 작성 전략 (케넨 전자책 기반)

블로그 초안 작성 시 아래 전략을 my-blog-style.md 말투 규칙과 함께 적용한다.

---

## 1. 글 쓰기 전 반드시 할 것

**메인 키워드를 먼저 네이버에 검색한다.**

- 어떤 글이 상위노출 되고 있는지 패턴 확인
- 제목 길이, 키워드 위치, 글 구조 파악
- 스마트블록 세부키워드 확인 (자동완성 켜기)

---

## 2. 제목 작성 규칙

### 스마트블록 제목 구성법
```
[메인키워드] + [스마트블록 세부키워드] + [부연 연관키워드]
예: 바르셀로나 가우디투어 후기 | 까사바트요, 구엘공원, 사그라다 파밀리아 (2025)
```

- 스마트블록 세부키워드는 자동완성/연관검색어에서 찾아서 넣기
- 제목 키워드 순서는 키워드마다 다름 → 상위노출 글 패턴과 일치시키기
- 여러 스마트블록 세부키워드를 제목에 동시에 넣으면 다중 노출 가능

### 좋은 제목 vs 나쁜 제목
| 나쁜 제목 | 좋은 제목 |
|---|---|
| 바르셀로나 가우디 / 사그라다 파밀리아 / 구엘공원 / 여행후기 | 바르셀로나 가우디투어 후기 까사바트요 구엘공원 하루 코스 |
| 여행 2일차 / 맛있었던 것들 / 좋았던 곳 | 마요르카 칼로 데스 모로 가는 법 주차 꿀팁 2025 |

- `/` 로 단어 나열하는 제목 금지 → 저품질 원인
- 제목에 관련 없는 키워드 혼합 금지 (알고리즘 혼란)
- 낚시성 제목 금지 (이탈률 상승 → 품질 하락)

---

## 3. 본문 구성 규칙

### 문단형 스니펫 방식 (필수)
- 소주제(인용구) 4개 이상 사용
- 각 소주제 아래 문단: 최소 3줄 ~ 최대 10줄
- "사진 - 글 1줄 - 사진 - 글 1줄" 패턴 금지

### 소주제 작성 규칙
- 간결하게 (긴 소주제 금지)
  - 나쁜 예: "스페셜 뇨끼 크림치즈 대파구이 아스파라거스 후기"
  - 좋은 예: "메뉴 / 외관 / 주차 / 분위기"
- 소주제에 같은 키워드 반복 금지
  - 나쁜 예: "신사역 맛집 분위기 / 신사역 맛집 서비스 / 신사역 맛집 후기"

### 핵심 내용 파악법
글을 쓰기 전 해당 키워드 상위노출 글 3개를 보고
공통으로 다루는 요소를 파악해서 내 글에도 반드시 포함

예) 여행 후기 키워드 공통 요소:
- 교통편 + 비용
- 입장료/예약 방법
- 실제 경험 + 팁
- 주의사항

---

## 4. 저품질 금지 사항

- 키워드 과다 반복 금지 (3회 이상 같은 단어 반복은 마이너스)
- 검색자 의도와 맞지 않는 내용 금지
  - 예) "풍향고 부다페스트" 검색자는 방송에 나온 장소 정보를 원하는 것
  - 거기에 여행 투어 상품 홍보 글을 얹으면 의도 불일치
- 일기 느낌의 주관적 감상 나열만 있는 글은 검색 누락 가능성 높음

---

## 5. 여행 블로그 특화 전략

**여행 한 번 = 여러 편으로 세분화해서 쓰기**

| 주제 | 예시 키워드 |
|---|---|
| 교통 | 바르셀로나 공항 시내 이동, R2N 타는 법 |
| 항공 | 라이언에어 수하물 규정, 비상구 좌석 후기 |
| 렌터카 | 마요르카 렌터카 추천, 위버 렌터카 후기 |
| 숙소 | 호텔명 후기, 지역명+호텔 추천 |
| 관광지 | 장소명 + 입장료/예약/꿀팁 |
| 맛집 | 식당명, 지역명+맛집 |

한 글에 모든 것을 우겨넣지 말고 주제별로 개별 글 작성

**맛집/식당 글 주의:**
- 맛 평가보다 기본 정보 중심 (위치, 메뉴, 가격, 영업시간, 주차)
- 부정적 평가 최소화 (알고리즘이 긍정 포스트 선호)
- 경험적 표현 활용: "친절해요", "신선해요", "가성비 좋아요", "주차 편해요"

---

## 6. 금액/수치 표기

- 반드시 화폐 단위 명확히 표기
  - 나쁜 예: "9.0유로", "5.5만"
  - 좋은 예: "9유로", "5만 5천원", "55,000원"
- 네이버 알고리즘이 구체적 금액 표기 있는 글을 상위노출

---

## 7. 유용한 키워드 분석 도구

- **키자드 (키워드마법사)**: 월간 검색량 + 발행 글 수 (수요/공급 비교)
- **키워드마스터**: 월간 검색량 분석
- **엔서포터**: 키워드 검색 시 자동으로 검색량 + 연관 키워드 분석
- **데이터랩툴즈 헬퍼**: 글자수 표시, 상위노출 글 분석
- **판다랭크**: 실시간 트렌드 + 블로그 분석

---

## 8. 키워드 진입 주의사항 (22장)

**자동완성에 있어도 피해야 하는 키워드:**
- 검색 시 최상단에 바로 답이 나오는 키워드 → 블로그까지 안 들어옴
  - 나쁜 예: "드라마 출연진", "영화 줄거리", "관객수", "몇부작"
  - 좋은 예: "드라마 원작", "해석", "결말 의미" (블로그에서 찾을 내용)
- 월간 검색량 100건 미만 키워드 → 하루 방문자 3명 수준

**초보자가 피해야 할 검색 영역:**
- 카페글 위주 영역
- 브랜드 콘텐츠(파워콘텐츠 광고) 영역
- 인플루언서 콘텐츠 영역
- 20대 인기글 영역

**특정 장소명 → 타겟 키워드로 전환:**

| 좁은 키워드 (피하기) | 넓은 타겟 키워드 (추천) |
|---|---|
| 이천 한국동요박물관 | 이천 가볼만한곳, 경기도 아이랑 |
| 바르셀로나 HG시티스위트 | 바르셀로나 숙소 추천, 바르셀로나 호텔 |
| 마요르카 Sa Franquesa Nova | 마요르카 숙소, 마요르카 렌터카 여행 |

→ 특정 장소 이름은 본문에 넣되, 제목 메인 키워드는 검색량 있는 타겟 키워드로

---

## 9. 쓰면 안 되는 글 (36장)

- 병원명 + 시술 전후 사진 조합 금지 (명예훼손 이슈)
- TV/방송 화면 캡처 이미지 사용 금지
- 상표권 등록 단어 사용 전 키프리스에서 확인
- AI가 지어낸 경험/메뉴/정보 그대로 올리는 것 금지

---

## 10. AI 활용법 (37장)

**AI가 잘하는 것 (활용):**
- 글 구조 잡기 (소주제 뼈대)
- 표현 보완, 문장 다듬기
- 상위노출 글 3개 복붙 → 공통점 분석 요청

**AI가 못하는 것 (직접 해야 함):**
- 제목 구성 → 반드시 직접 (네이버 알고리즘 미반영)
- 소주제 키워드 배치 → 직접
- 실제 경험 부분 → 직접 다듬기

**AI 초안 블로그 붙여넣기 주의:**
- 반드시 txt 메모장에 한 번 붙였다가 복사 → 히든 소스 제거
- 애플 메모장 X → 카카오톡 나에게 쓰기 활용 가능
- 유료 버전 사용 필수 (무료 버전 오류 많음)

**상위노출 글 분석 프롬프트 활용법:**
1. 원하는 키워드 상위노출 글 3개 복사 → 메모장 저장
2. GPT에 "공통점 분석해줘" 요청
3. 분석 결과 바탕으로 내 글 구조 설계
4. 제목/소주제는 직접 구성, 본문 초안만 AI 활용

---

## 11. 초안 작성 시 체크리스트

글 작성 전:
- [ ] 메인 키워드 먼저 네이버 검색
- [ ] 스마트블록 세부키워드 확인
- [ ] 상위노출 글 3개 패턴 파악
- [ ] 검색자 의도 파악 (정보형? 후기형? 트렌드형?)

글 작성 중:
- [ ] 제목에 스마트블록 키워드 포함
- [ ] 소주제 4개 이상 (인용구 형태)
- [ ] 소주제별 3~10줄 문단
- [ ] 금액은 화폐단위 명확히
- [ ] 키워드 반복 3회 이상 금지
- [ ] / 나열형 제목/소주제 금지
