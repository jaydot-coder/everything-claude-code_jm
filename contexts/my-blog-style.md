# 블로그 말투 규칙

블로그 글 작성 시 반드시 따르는 규칙. 샘플(`my-blog-samples.md`)과 충돌 시 이 파일이 우선한다.

---

## AI 말투 금지 패턴

| 금지 패턴 | 대체 방향 |
|-----------|-----------|
| "숨겨진 보석", "동화 같은", "그림 같은 분위기를 자아냅니다" | 직접적 묘사로 대체 ("물 색깔이 진짜 투명해요") |
| "안녕하세요! 곰키입니다" 같은 인사말로 글 시작 | 해당 일차/상황으로 자연스럽게 바로 시작 |
| "탄성이 절로 나옵니다", "감동이었습니다" | 구체적 경험 서술로 대체 |
| "솔직하게 정리했습니다" | 그냥 내용으로 들어가기 |
| "끝까지 읽어주세요!" | 삭제 |
| "이 두 곳만큼은 꼭 가보시길 추천합니다" | "여기는 진짜 가볼 만했어요" 수준으로 |
| "~분들께 도움이 될 내용이 많으니" | 삭제 |
| "알고 계셨나요?" | 그냥 서술로 ("가우디가 개조한 흔적이 있어요") |
| 글 끝 해시태그 | 사용하지 않음 (네이버 SEO 실효 없음) |
| "이 코스가 정답이라고 감히 말씀드립니다" | 직접 평가 ("이 동선이 제일 효율적이었어요") |
| "강력 추천합니다 / 권장합니다" 반복 | "~하는 게 좋아요 / 낫습니다" 정도로 |
| 도입부 "이번 편은 ~, 이번 여행의 하이라이트~" 매번 동일 구조 | 그날 상황이나 인상적인 포인트로 자연스럽게 시작 |
| "이번 여행에서 가장 ~한 시간 중 하나였습니다" | 삭제하거나 구체적 표현으로 대체 |
| "이 글 미리 읽고 가세요" | 삭제 |

---

## 말투 특징 요약

- **어투**: 해요체/합니다체 혼합, 반격식
- **문장 길이**: 짧게 끊기, 한 단락 1~3문장
- **평가 방식**: 직접적 ("진짜 좋았어요", "생각보다 괜찮았습니다", "이건 별로였어요")
- **부연 설명**: 괄호 활용 자주 ("(여기서 주의)", "(미리 챙겨야 함)")
- **감정 표현**: 과장 없이 담담하게, 구어체로 ("진짜 꿀이었습니다", "당황했어요")
- **정보 전달**: 경험 + 팁을 자연스럽게 섞음, 체크리스트/표 활용
- **마무리**: 다음 편 예고 + 댓글 유도 간단하게 (1줄 수준)
- **글 시작**: 인사말 없이, 해당 주제/상황으로 바로 시작
