# Naver Blog Rules

네이버 블로그 글 작성 시 반드시 따라야 하는 포맷 및 최적화 규칙.

## 글 구조

### 제목

- 핵심 키워드를 제목 앞쪽에 배치
- 25자 이내 권장 (네이버 검색 결과에서 잘리지 않도록)
- 숫자, 연도, 구체적 정보 포함 시 클릭률 상승
- 예: "제주도 3박4일 여행코스 추천 (2026년 최신)"

### 본문 구조

```
[도입부] - 1~3문장으로 글의 주제와 읽어야 할 이유
[이미지 위치]
[소제목 1] - 핵심 내용
[이미지 위치]
[소제목 2] - 세부 내용
[이미지 위치]
[소제목 3] - 추가 정보 / 팁
[마무리] - 요약 또는 개인 의견
[태그]
```

### 소제목

- 본문에 소제목 2~4개 사용
- 소제목에도 키워드 포함
- `■`, `▶`, `●` 등의 기호 또는 볼드체로 구분

### 구분선

- 섹션 사이에 구분선 사용: `---` 또는 빈 줄 2개
- 과도한 구분선 사용 지양

## 이미지 가이드

- 글 상단(도입부 직후)에 대표 이미지 1장
- 소제목마다 관련 이미지 1~2장
- 이미지 위치는 `[이미지: 설명]` 형태로 표시
- 총 이미지 5~10장 권장

## 글 길이

- 최소: 800자 (네이버 검색 노출 기준)
- 권장: 1500~3000자
- 최대: 5000자 (너무 길면 이탈률 증가)
- 모바일 가독성을 위해 한 단락 3~4문장 이내

## SEO 최적화

### 키워드 배치

- 제목에 주요 키워드 1회
- 본문 첫 문단에 주요 키워드 1회
- 소제목에 관련 키워드 자연스럽게 포함
- 본문 전체에 키워드 3~5회 (자연스럽게, 억지 반복 금지)

### 첫 문단 SEO 훅 (필수)

- 첫 2~3문장 안에 반드시 핵심 키워드 + 글의 핵심 요약을 포함
- 인사말/근황으로 시작하지 않음 — 키워드가 담긴 문장으로 바로 진입
- 네이버 검색 결과의 미리보기(스니펫)에 첫 문단이 노출되므로 클릭을 유도하는 문장으로 구성
- 예 (BAD): "안녕하세요! 곰키입니다 오랜만에 글 올리네요~"
- 예 (GOOD): "스페인 여행 준비물, 14박 16일 직접 다녀오며 검증한 체크리스트를 공유합니다"

### 롱테일 키워드 전략

- 주요 키워드에 연도/지역/수량 등 구체적 수식어를 조합
- 예: "스페인 여행" → "2026 스페인 여행 준비물", "바르셀로나 소매치기 방지"
- 본문 내에 롱테일 키워드를 2~3개 자연스럽게 분산 배치
- 사용자가 실제로 검색할 법한 구어체 키워드도 포함 (예: "스페인 뭐 챙겨가야 하나")

### 이미지 설명 키워드

- `[이미지: 설명]` 표시 시 설명에 반드시 검색 키워드 포함
- 네이버 이미지탭 검색 노출에 직접 영향
- 예 (BAD): `[이미지: 사진]`, `[이미지: AI 활용 설정]`
- 예 (GOOD): `[이미지: 스페인 여행 도난방지 힙색 착용 모습]`

### 내부링크 & 시리즈 연결

- 관련 주제의 이전/다음 포스팅 링크를 본문 내에 자연스럽게 삽입
- 시리즈 글일 경우 도입부 또는 마무리에 시리즈 목차 안내
- 체류시간 증가 + 재방문 유도 효과
- 예: "이심 구매 방법은 이전 포스팅에서 자세히 다뤘으니 참고해주세요 (링크)"

### CTA (Call To Action)

- 마무리 부분에 독자 행동 유도 문구 1~2개 포함
- 댓글 유도: "궁금한 점은 댓글로 남겨주세요"
- 공감 유도: "도움이 되셨다면 공감 한번 부탁드려요"
- 이웃 유도: "여행 꿀팁 시리즈 계속 올릴 예정이니 이웃추가 해두시면 편해요"
- 과도한 CTA는 지양 — 1~2개만 자연스럽게

## 포맷 규칙

### 텍스트 포맷

- 네이버 블로그 에디터에 복사-붙여넣기 가능한 순수 텍스트
- HTML 태그 사용하지 않음 (에디터가 자동 처리)
- 볼드 강조는 `**텍스트**` 대신 직접 표시: 텍스트 앞뒤에 설명 추가
- 줄바꿈은 빈 줄 하나로 처리

### 금지 사항

- 저작권 있는 이미지 직접 삽입 안내 금지
- 허위/과장 광고성 문구 금지
- 타 블로그 글 복사 금지
- 의학/법률 전문 조언 형태의 글 금지 (면책 문구 필수)

## 모바일 최적화

- 네이버 블로그 트래픽의 70%+ 가 모바일
- 한 줄에 15~20자 내외가 읽기 편함
- 긴 문장보다 짧은 문장 여러 개가 가독성 좋음
- 중요 정보는 스크롤 없이 보이는 상단에 배치
